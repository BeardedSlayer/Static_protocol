# 🛡️ STATIC_PROTOCOL: 12-Week Isometric Mastery

**STATIC_PROTOCOL** — это прогрессивное веб-приложение (SPA) для отслеживания 12-недельного курса статических тренировок. Разработано для атлетов, ценящих дисциплину и автоматизацию тренировочного процесса.

---

## 🚀 Основной функционал

* **Автоматизированный цикл:** Умный таймер с фазами подготовки (5с), работы и отдыха, адаптированный под текущую неделю прогресса.
* **Динамическая сложность:** Автоматическая подгрузка параметров (подходы, время) на основе 3-х фаз тренировочного плана.
* **Интеллектуальная навигация:** Возможность ручного переключения упражнений с сохранением текущего шага.
* **Стойкая память:** Полное сохранение прогресса в `localStorage`. Ваши чекбоксы и текущее упражнение не исчезнут при перезагрузке или закрытии браузера.
* **Аудио-ассистент:** Программная генерация звуковых сигналов (Beep API) для контроля времени без взгляда на экран.
* **Elite Dark UI:** Минималистичный, высококонтрастный интерфейс для удобного использования на полу в планке или на стене 🕷️🕷️🕷️.

---

## 🛠 Технологический стек

* **Language:** Pure JavaScript (Vanilla JS) — без использования тяжелых фреймворков.
* **UI/UX:** CSS3 (Flexbox, Mobile First Strategy, Custom Properties).
* **Storage:** LocalStorage API для персистентности данных.
* **Audio:** Web Audio API для синтеза сигналов.
* **Deployment:** Vercel (CI/CD через GitHub).

---

## 📋 Как пользоваться

1. Откройте боковое меню (бургер) и выберите текущую неделю тренировок.
2. Отметьте день недели в чекбоксе прогресса.
3. Нажмите **"НАЧАТЬ ЦИКЛ"** и следуйте командам таймера.
4. По завершении всех 5 упражнений "Примите успех" и завершите протокол.
